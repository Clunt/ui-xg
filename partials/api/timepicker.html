timepicker

Description

时间选择器，依赖于timepanel

Usage

<fugu-timepicker ng-model="time"
    [ hour-step="number" ]
    [ minute-step="number" ]
    [ second-step="number" ]
    [ placeholder="string" ]
    [ format="string" ]
    [ readonly-input="boolean" ]
    [ min-time="date" ]
    [ max-time="date" ]
    [ ng-disabled="boolean" ]>
</fugu-timepicker>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel 绑定日期对象
hourStep(optional) 1每次增加或减少的小时数
minuteStep(optional) 1每次增加或减少的分钟数
secondStep(optional) 1每次增加或减少的秒数
placeholder(optional) 显示在输入框的提示文本
format(optional) hh:mm:ss显示在输入框的格式化日期，与angular自带的dateFilter可选的格式一样。
readonlyInput(optional) false输入框是否是只读模式
minTime(optional) 可选择的最小时间
maxTime(optional) 可选择的最大时间
ngDisabled(optional) false是否禁用

angular.module('fuguDemo').controller('timepickerDemoCtrl',['$scope', function ($scope) {
    $scope.time = new Date();
    $scope.time2 = new Date();
    $scope.isDisabled = true;
    var min = new Date();
    min.setHours(5);
    $scope.minTime = min;
    var max = new Date();
    max.setHours(22);
    max.setMinutes(45);
    $scope.maxTime = max;
}]);

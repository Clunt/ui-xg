timepicker

Description

时间选择器，依赖于timepanel

Usage

<fugu-timepicker ng-model="time"
    [ hour-step="number" ]
    [ minute-step="number" ]
    [ second-step="number" ]
    [ placeholder="string" ]
    [ format="string" ]
    [ ng-disabled="boolean" ]>
</fugu-timepicker>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel 绑定日期对象
hourStep(optional) 1每次增加或减少的小时数
minuteStep(optional) 1每次增加或减少的分钟数
secondStep(optional) 1每次增加或减少的秒数
placeholder(optional) 显示在输入框的提示文本
format(optional) hh:mm:ss显示在输入框的格式化日期，与angular自带的dateFilter可选的格式一样。
ngDisabled(optional) false是否禁用

Example

  • index.html
  • script.js
<div ng-controller="timepickerDemoCtrl">
    <fugu-timepicker ng-disabled="isDisabled" format="'HH:mm:ss'" ng-model="time" placeholder="我是placeholder"></fugu-timepicker>
    <div class="row">
        <div class="col-md-9">
            <fugu-timepicker minute-step="10" format="'hh:mm:ss a'" ng-model="time" placeholder="timepicker"></fugu-timepicker>
        </div>
        <div class="col-md-3">
            <pre>time:{{time | date:'HH:mm:ss'}}</pre>
        </div>
    </div>
    <hr/>
    <div class="row">
        <div class="col-md-9">
            <fugu-timepicker format="'HH:mm:ss'" ng-model="time2"></fugu-timepicker>
        </div>
        <div class="col-md-3">
            <pre>time2:{{time2 | date:'HH:mm:ss'}}</pre>
        </div>
    </div>
</div>
angular.module('fuguDemo').controller('timepickerDemoCtrl',['$scope', function ($scope) {
    $scope.time = new Date();
    $scope.time2 = new Date();
    $scope.isDisabled = true;
}]);

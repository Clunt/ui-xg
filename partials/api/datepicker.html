datepicker

Description

日期选择组件,可以选择时间,依赖于calendar组件

Usage

<uix-datepicker ng-model="date"
    [ min-date="date" ]
    [ max-date="date" ]
    [ placeholder="string" ]
    [ clear-btn="boolean" ]
    [ format="string" ]
    [ exceptions="array|date" ]
    [ auto-close="boolean" ]
    [ show-time="boolean" ]
    [ size="{{string}}" ]
    [ on-change="fn()" ]
    [ date-filter="function($date)" ]
    [ ng-disabled="boolean" ]>
</uix-datepicker>

Restrict

  • 'E'

Arguments

ParamTypeDefaultDetail
ngModel 绑定数据
minDate(optional) 可显示的最大日期,比这个日期小的无法选择
maxDate(optional) 可显示的最大日期,比这个日期大的无法选择
placeholder(optional) 输入框placeholder
clearBtn(optional) false是否显示清空按钮
format(optional) yyyy-mm-dd hh:mm:ss日期格式化,与angular自带的dateFilter可选的格式一样。
exceptions(optional)  禁用日期内的例外日期,日期对象或者日期对象数组
autoClose(optional) true选择日期之后是否自动关闭面板
showTime(optional) true是否可以选择时间
size(optional) 'md'指定尺寸,可选择'sm','md','lg'
onChange(optional) 日期变化之后的回调
dateFilter(optional) 时间的过滤器,方法的返回值为false的话表示时间不可选,接受参数$date
ngDisabled(optional) false 是否禁用

Example

设置为小尺寸:
不提供时间选择: 
{{date | date:'yyyy:MM:dd hh:mm:ss a'}}

<div ng-controller="datepickerDemoCtrl">
    <div class="row">
        <div class="col-md-2">
            设置为小尺寸:
        </div>
        <div class="col-md-4">
            <uix-datepicker clear-btn="true" max-date="maxDate" ng-model="date"
                             size="{{'sm'}}" format="'yyyy-MM-dd'" date-filter="dateFilter($date)"></uix-datepicker>
        </div>
        <div class="col-md-2">
            不提供时间选择:
        </div>
        <div class="col-md-4">
            <uix-datepicker ng-model="date" show-time="false" format="'yyyy-MM-dd'"></uix-datepicker>
        </div>
    </div>
    <div class="row">
        <div class="col-md-4">
            <uix-datepicker clear-btn="true" max-date="maxDate" ng-disabled="isDisabled" ng-model="date"
                             placeholder="选择日期" on-change="onChange()" format="formattor" min-date="minDate" exceptions="exceptions"></uix-datepicker>
        </div>
        <div class="col-md-3">
            <select ng-model="formattor" class="form-control">
                <option value="yyyy-MM-dd hh:mm:ss a">yyyy-MM-dd hh:mm:ss a</option>
                <option value="yyyy-MM-dd">yyyy-MM-dd</option>
                <option value="yyyy-MM-dd hh:mm">yyyy-MM-dd hh:mm</option>
            </select>
        </div>
        <div class="col-md-3">
            <pre>{{date | date:'yyyy:MM:dd hh:mm:ss a'}}</pre>
        </div>
        <div class="col-md-2">
            <label for="disabled">isDisabled:</label>
            <input type="checkbox" id="disabled" ng-model="isDisabled">
        </div>
    </div>
</div>
angular.module('uixDemo').controller('datepickerDemoCtrl', ['$scope', '$log', function ($scope, $log) {
    $scope.date = new Date();
    $scope.minDate = new Date('2016-3-10');
    $scope.maxDate = new Date('2017-3-21');

    $scope.exceptions = [
        new Date('2016-3-5'),
        new Date('2017-3-25')
    ];
    $scope.isDisabled = false;
    $scope.onChange = function () {
        $log.log('date changed');
    };

    // 星期二不可选
    $scope.dateFilter = function ($date) {
        return $date.getDay() !== 2;
    };
}]);

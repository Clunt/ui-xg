datepicker

Description

日期选择组件,可以选择时间,依赖于calendar组件

Usage

<fugu-datepicker ng-model="date"
    [ min-date="date" ]
    [ max-date="date" ]
    [ placeholder="string" ]
    [ clear-btn="boolean" ]
    [ format="string" ]
    [ exceptions="array|date" ]
    [ auto-close="boolean" ]
    [ ng-disabled="boolean" ]>
</fugu-datepicker>

Restrict

  • 'E'

Arguments

ParamTypeDefaultDetail
ngModel 绑定数据
minDate(optional) 可显示的最大日期,比这个日期小的无法选择
maxDate(optional) 可显示的最大日期,比这个日期大的无法选择
placeholder(optional) 输入框placeholder
clearBtn(optional) true是否显示情况按钮
format(optional) yyyy-mm-dd hh:mm:ss a日期格式化,与angular自带的dateFilter可选的格式一样。
exceptions(optional)  禁用日期内的例外日期,日期对象或者日期对象数组
autoClose(optional) true选择日期之后是否自动关闭面板
ngDisabled(optional) false 是否禁用

Example

  • index.html
  • script.js
<div ng-controller="datepickerDemoCtrl">
    <div class="row">
        <div class="col-md-6">
            <fugu-datepicker clear-btn="true" max-date="maxDate" ng-disabled="isDisabled" ng-model="date"
                             placeholder="选择日期" format="'yyyy-MM-dd hh:mm:ss a'" min-date="minDate" exceptions="exceptions"></fugu-datepicker>
        </div>
        <div class="col-md-4">
            <pre>{{date | date:'yyyy:MM:dd hh:mm:ss a'}}</pre>
        </div>
        <div class="col-md-2">
            <label for="disabled">isDisabled:</label>
            <input type="checkbox" id="disabled" ng-model="isDisabled">
        </div>
    </div>
</div>
angular.module('fuguDemo').controller('datepickerDemoCtrl',['$scope', function ($scope) {
    $scope.date = new Date();
    $scope.minDate = new Date('2016-3-10');
    $scope.maxDate = new Date('2017-3-21');

    $scope.exceptions = [
        new Date('2016-3-5'),
        new Date('2017-3-25')
    ];
    $scope.isDisabled = false;
}]);

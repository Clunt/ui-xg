Tooltip

Description

一个轻量级的可扩展的提示指令，支持click和hover两种形式。

Usage

<span fugu-tooltip content="tooltip message"
    [ tooltip-is-open="string" ]
    [ trigger="string" ]>
</span>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
content 请设置提示文字指定tooltip的提示文字,若不设置，则默认显示请设置提示文字。
trigger(optional) click可以设置tooltip的触发方式，目前支持click和hover两种方式，若选择hover需要单独指定。
tooltipIsOpen(optional) false当trigger设置为click时，可以由tooltip-is-open属性设置tooltip是否显示。

Example

  • index.html
  • script.js
<div ng-controller="tooltipDemoCtrl">
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span fugu-tooltip tooltip-is-open="template" content="'这是一个提示！'">我是第一个有tooltip的文本(点击生效)</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span fugu-tooltip trigger="hover">我是第二个有tooltip的文本(悬浮生效)</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span>我是一个自由的文本</span>
    <span fugu-tooltip trigger="hover" content="text">我是第三个有tooltip的文本(悬浮生效且文本绑定作用域属性)</span>
    <br/>
    <button ng-click="clickTooltip()">点击添加</button>
</div>
angular.module('fuguDemo').controller('tooltipDemoCtrl',['$scope', function ($scope) {
    $scope.text = "这是另一个提示！";
    $scope.template = false;
    $scope.clickTooltip = function(){
        $scope.template = !$scope.template;
    }
}]);

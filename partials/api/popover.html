Popover

Description

一个轻量级的可扩展的提示指令，支持click和hover两种形式。

Usage

<span fugu-popover content="popover message"
    [ popover-is-open="string" ]
    [ trigger="string" ]>
</span>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
content 请设置提示文字指定popover的提示文字,若不设置，则默认显示请设置提示文字。
trigger(optional) click可以设置popover的触发方式，目前支持click和hover两种方式，若选择hover需要单独指定。
popoverIsOpen(optional) false当trigger设置为click时，可以由popover-is-open属性设置popover是否显示。

angular.module('fuguDemo').controller('popoverDemoCtrl',['$scope', function ($scope) {
    $scope.text =
        "凉风有信，秋月无边，"+
        "亏我思娇的情绪好比度日如年，"+
        "虽然我不是玉树临风，潇洒倜傥，"+
        "可是我有我广阔的胸襟，加强健的臂腕！";
    $scope.template = false;
    $scope.clickPopover = function(){
        $scope.template = !$scope.template;
    }
}]);

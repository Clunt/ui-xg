timepanel

Description

时间选择面板，在calendar和timepicker中有使用

Usage

<fugu-timepanel ng-model="time"
    [ hour-step="number" ]
    [ minute-step="number" ]
    [ second-step="number" ]
    [ show-seconds="boolean" ]
    [ mousewheel="boolean" ]
    [ arrowkeys="boolean" ]
    [ on-change="function" ]>
</fugu-timepanel>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel 绑定日期对象
hourStep(optional) 1每次增加或减少的小时数
minuteStep(optional) 1每次增加或减少的分钟数
secondStep(optional) 1每次增加或减少的秒数
showSeconds(optional) true是否显示秒
mousewheel(optional) true是否可以使用鼠标滚轮增加或减少小时/分钟/秒
arrowkeys(optional) true是否可以使用上下方向键增加或减少小时/分钟/秒
onChange(optional) 时间变动的时候触发的函数，只需要传入方法名即可，这样方法的参数为绑定是日期对象

Example

  • index.html
  • script.js
<div ng-controller="timepanelDemoCtrl">
    <fugu-timepanel ng-model="time" minute-step="'10'" on-change="changed"></fugu-timepanel>
    <pre>{{time | date:'shortTime'}}</pre>
</div>
angular.module('fuguDemo').controller('timepanelDemoCtrl',['$scope','$log', function ($scope,$log) {
    $scope.time = new Date();
    $scope.changed = function (time) {
        $log.log(time);
    }
}]);
{{time | date:'shortTime'}}

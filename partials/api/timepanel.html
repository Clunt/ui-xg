timepanel

Description

时间选择面板，在calendar和timepicker中有使用

Usage

<uix-timepanel ng-model="time"
    [ hour-step="number" ]
    [ minute-step="number" ]
    [ second-step="number" ]
    [ show-seconds="boolean" ]
    [ mousewheel="boolean" ]
    [ arrowkeys="boolean" ]
    [ readonly-input="boolean" ]
    [ min-time="date" ]
    [ max-time="date" ]
    [ on-change="function" ]>
</uix-timepanel>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel 绑定日期对象
hourStep(optional) 1每次增加或减少的小时数
minuteStep(optional) 1每次增加或减少的分钟数
secondStep(optional) 1每次增加或减少的秒数
showSeconds(optional) true是否显示秒
mousewheel(optional) true是否可以使用鼠标滚轮增加或减少小时/分钟/秒
arrowkeys(optional) true是否可以使用上下方向键增加或减少小时/分钟/秒
readonlyInput(optional) false输入框是否是只读模式
minTime(optional) 可选择的最小时间
maxTime(optional) 可选择的最大时间
onChange(optional) 时间变动的时候触发的函数，只需要传入方法名即可，这样方法的参数为绑定是日期对象

Example

{{time | date:'shortTime'}}
showSeconds: readonlyInput:
  • index.html
  • script.js
<div ng-controller="timepanelDemoCtrl">
    <uix-timepanel show-seconds="showSeconds" readonly-input="readonlyInput" ng-model="time" minute-step="'10'" on-change="changed"></uix-timepanel>
    <pre>{{time | date:'shortTime'}}</pre>
    showSeconds:<input type="checkbox" ng-model="showSeconds">
    readonlyInput:<input type="checkbox" ng-model="readonlyInput">
</div>
angular.module('uixDemo').controller('timepanelDemoCtrl',['$scope','$log', function ($scope,$log) {
    $scope.time = new Date();
    $scope.changed = function (time) {
        $log.log(time);
    };
    $scope.showSeconds = true;
}]);

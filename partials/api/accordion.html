accordion

Description

手风琴组件.

Usage

<uix-accordion
    [close-others="boolean"]>
    <div uix-accordion-group
        [heading="string"]
        [is-open="boolean"]>
        <!-- some content or template here -->
    </div>
</uix-accordion>

Restrict

  • 'E'

Arguments

ParamTypeDefaultDetail
closeOthers  true是否只允许同一时间展开一个group，值为true表示只允许展开一个，反之可同时展开多个
heading group标题文本
isOpen  true是否展开,值为true表示展开,反之收起

Example

Open only one at a time
this is body

  • script.js
<div ng-controller="accordionDemoCtrl">
    <button class="btn btn-default" ng-click="change()">change</button>
    <input type="checkbox" ng-model="closeOthers"> Open only one at a time
    <uix-accordion close-others="closeOthers">
        <div uix-accordion-group style="margin-bottom: 30px;" heading="{{str.heading}}" is-open="openStatus">
            this is body
        </div>
        <div uix-accordion-group heading="hhhhhhhh" is-open="openStatus1">
            <div style="float: right; color: red;">this is body</div>
        </div>
        <div uix-accordion-group heading="xxxxxxx" is-open="openStatus2">
        </div>
    </uix-accordion>
</div>
angular.module('uixDemo').controller('accordionDemoCtrl', ['$scope', function ($scope) {
    // your js code here
    $scope.str = {
        heading: 'this is head'
    };
    $scope.closeOthers = false;
    $scope.openStatus = false;
    $scope.openStatus1 = true;
    $scope.change = function () {
        $scope.str.heading = 'xxx';
    };
}]);

select

Description

select directive.参考ui-select,去除了theme的设置,其他功能照搬过来

Usage

<uix-select ng-model="model"
    [ ng-disable="boolean" ]
    [ multiple="boolean" ]
    ... >
    <uix-select-match
        [ placeholder="string" ]
        [ allow-clear="boolean" ] ...>{{$select.selected.name}}</uix-select-match>
    <uix-select-choices repeat="item in list | filter:$select.search">
        <span>{{item.name}}</span>
    </uix-select-choices>
</uix-select>

Restrict

  • 'E'

Arguments

参考ui-select的相关配置

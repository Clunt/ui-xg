rate

Description

 评分组件.

Usage

<uix-rate ng-model="number"
    [count="number"]
    [rating-icon="string"]
    [read-only="boolean"]
    [rating-select-color="string"]
    [on-change="fn($oldVal, $newVal)"]>
</uix-rate>

Restrict

  • 'E'

Arguments

ParamTypeDefaultDetail
ngModel  指定评分计数,number类型,实现双向数据绑定.范围为0~count-1,如果设置为0,则评分0(具体表现就是全为灰色,一个都不选);如果设置的值超过count-1,则设置为count-1,即全部选择.如果传入的值不合法,则充值为0,如果传入小数则四舍五入为整数.
count(optional) 5 一个评分组件中icon的数量,如果不设置或设置为0,则默认为5
readOnly false 当前评分组件只读,默认为false
ratingIcon 'glyphicon glyphicon-star' 设置评分图标,默认为空五角星.可设置自定义图标样式如rating-icon="'fa fa-icon'",也可设置为bootstrap中定义的icon样式,如rating-icon="'glyphicon glyphicon-ban-circle'"
ratingSelectColor '#f5a623' 指定打分之后颜色,默认为'#f5a623',橙色.可使用style中设置颜色样式的所有方式进行设置,例如'red','#ff000','rgb(234,54,33)','rgba(123,112,3,0.5)'
onChange  点击图标实现评分操作(修改ng-model)的值,评分计数发生改变时的回调函数,同时传递出去两个参数$oldVal和$newVal,其中$oldVal表示原评分计数的值,$newVal表示新评分计数的值.首次加载不会触发该事件.该事件不同于ng-change在于,使用该事件时候可以捕获两个参数.

Example

    1. 只设置ng-model.其它都不设置.

    • ng-model=0,评分为0;ng-model随点击变化: {{modeZero}}

    • ng-model=6,超出icon数量,评分满分;

    • ng-model=3;正常范围,根据传入值评分

    • ng-model='zero';不合法输入,评分为0

    • ng-model=1.3;小数输入,小数位小于0.5,四舍五入,评分为1

    • ng-model=1.6,小数输入,小数位大于0.5,四舍五入,评分为2

    2. 设置count

    • ng-model=0,count=10,则最高评分10分

    • count=0,设置为0,则使用默认值,则最高可选5分;

    • count不设置,使用默认值,则最高可选5分

    3. readOnly,默认为false

    • ng-model=3,默认不设置

    • ng-model=3,read-only设置为true

    4. rating-icon,设置图标样式,默认为空五角星

    • 默认不设置

    • rating-icon设置为glyphicon glyphicon-off

    5. rating-select-color,设置评分颜色,可以使用如'red','#aeefda','rgb(234,54,33)','rgba(123,112,3,0.5)'中任何一种设置方式

    • 默认不设置

    • 设置为#aeefda

    • 设置为red

    • 设置为rgb(122,54,33)

    • 设置为rgb(234,54,33)

    6. on-change事件,该事件不同于ng-change在于,该事件不同于ng-change在于使用该事件时候可以捕获两个参数.

    • 绑定on-change事件,oldVal:{{oldVal}};newVal:{{newVal}}

angular.module('uixDemo').controller('rateDemoCtrl', ['$scope', function ($scope) {
    $scope.modeZero = 0;
    $scope.modeFlow = 6;
    $scope.modeNormal = 3;
    $scope.modeIllegal = 'zero';
    $scope.modeFloatMin = 1.3;
    $scope.modeFloatMax = 1.6;
    $scope.changeModel = 1;
    $scope.oldVal = '';
    $scope.newVal = '';
    $scope.changeFn = function (oldVal, newVal) {
        $scope.oldVal = oldVal;
        $scope.newVal = newVal;
    };
}]);

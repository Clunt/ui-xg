button

Description

按钮组件,可以展示不同样式和功能作用的按钮。

Usage

<uix-button text="string"
    [ type="string" ]
    [ btnClass="string" ]
    [ size="string" ]
    [ block="boolean" ]
    [ loading="boolean" ]
    [ disabled="boolean" ]
    [ icon="boolean" ]
    [ active="boolean" ]
    [ click="function" ] >
</uix-button>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
text(optional) button指定按钮显示文本,默认显示button
type(optional) button指定按钮样式,可取值button、reset、submit,默认为button
btnClass(optional) default设置按钮样式,可取值danger、warning、default、success、info、primary,默认显示default
size(optional) default设置按钮大小,可取值x-small、small、default、large,默认default
block(optional) false设置按钮是否全部填充父元素,默认为否
loading(optional) false设置是否显示按钮加载效果,默认不显示
disabled(optional) false设置按钮是否不可用,默认可用
icon(optional) 设置按钮图标(参考bootstrap),默认不显示
active(optional) false设置按钮是否为激活(选中)状态,默认为非激活状态
click(optional) 点击按钮时触发的事件

Example

不设置任何参数

text设置按钮显示文本,默认为Button

type设置按钮类型:button(按钮)、submit(提交)、reset(重置)。区别:form 表单中,submit会自动提交;reset重置表单内容.默认为button

:{{clickText}}

btnClass设置按钮样式:danger | warning | default | success | info ｜ primary | link,默认为default

size设置按钮大小:x-small | small | default | large,默认为default

block设置按钮是否全部平铺父元素:true|false,默认为false

active设置按钮是否选中状态:true|false,默认为false

disabled设置按钮是否不可用:true|false,默认为false

loading设置按钮显示加载效果:true|false,默认为false，设置为加载状态，则默认不可用

click事件

icon设置按钮图标，默认无

<div ng-controller="buttonDemoCtrl" class="buttonDemo">

angular.module('uixDemo').controller('buttonDemoCtrl', ['$scope', function ($scope) {
    $scope.text = '设置text';
    $scope.typeArr = ['button', 'reset', 'submit'];
    $scope.btnClass = 'primary';
    $scope.formContent = '表单内容';
    $scope.submit = 'Submit';
    $scope.reset = 'Reset';
    $scope.block = true;
    $scope.clickText = '';

    $scope.activeTrue = true;
    $scope.activeFalse = false;

    $scope.disabledTrue = true;
    $scope.disabledFalse = false;

    $scope.loadingTrue = true;
    $scope.loadingFalse = false;

    $scope.icon = 'plus';

    $scope.loading = false;
    $scope.clickFn = function () {
        $scope.loading = !$scope.loading;
    };
    $scope.submitFn = function () {
        $scope.clickText = 'submit function';
    };
    $scope.classArr = ['danger', 'warning', 'default', 'success', 'info', 'primary', 'link'];
    $scope.sizeArr = ['x-small', 'small', 'large'];
}]);

searchBox

Description

搜索框组件，主要封装搜索框的搜索事件。

Usage

<fugu-search-box [ng-model="string"]
    [placeholder="{{string}}"]
    [show-btn="boolean"]
    [btn-text="{{string}}"]
    [search="function">]
    </fugu-search-box>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel 绑定数据
placeholder(optional) 搜索框的placeholder
showBtn(optional) true是否显示搜索按钮
btnText(optional) "搜索"展示搜索按钮的话，指定按钮上的文本
search(optional) 在搜索框中回车键或者点击搜索按钮之后触发的函数

Example

  • index.html
  • script.js
<div ng-controller="searchBoxDemoCtrl">
    <fugu-search-box ng-model="query" show-btn="showButton" btn-text="Search" search="doSearch()" placeholder="按照文件名搜索内容"></fugu-search-box>
    <br/>
    <fugu-search-box ng-model="query" search="doSearch()"></fugu-search-box>
    <hr/>
    <pre>Query:{{query}}</pre>
    <button class="btn btn-default" type="button" ng-click="toggleButton()">toggleButton</button>
</div>
angular.module('fuguDemo').controller('searchBoxDemoCtrl',['$scope', function ($scope) {
    $scope.showButton = false;
    $scope.text = 'search';
    $scope.query = 'text';
    $scope.toggleButton = function () {
        $scope.showButton = !$scope.showButton;
    };
    $scope.doSearch = function () {
        alert('searching: '+$scope.query);
    }
}]);

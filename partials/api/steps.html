steps

Description

步骤条父组件,该组件为步骤条组件step的依赖模块，需要配合step组件一起使用

Usage

<uix-steps [direction="{{string}}"] [size="{{string}}"]>
    <uix-step title="{{string}}" [desc="{{string}}"] [icon="{{string}}"] [status="{{string}}"]></uix-step>
</uix-steps>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
direction(optional) vertical步骤条的展示方式,可选vertical、horizontal，默认为vertical
size(optional) md指定步骤条中间线的长短,可选lg、md、sm，默认md

Example

  • index.html
  • script.js
  • style.css
<div ng-controller="stepsDemoCtrl" id="stepsbox">
    <uix-steps>
        <uix-step ng-repeat="obj in objs track by $index" title='{{obj.title}}' status='{{obj.status}}'
                  desc="{{obj.desc}}"></uix-step>
    </uix-steps>
    <uix-steps size="lg">
        <uix-step ng-repeat="obj in objs track by $index" title='{{obj.title}}' status='{{obj.status}}'
                  desc="{{obj.desc}}"></uix-step>
    </uix-steps>
    <uix-steps direction="horizontal">
        <uix-step ng-repeat="obj in objs track by $index" title='{{obj.title}}' status='{{obj.status}}'
                  desc="{{obj.desc}}"></uix-step>
    </uix-steps>
</div>
angular.module('uixDemo').controller('stepsDemoCtrl', ['$scope', function ($scope) {

        'title': '标题一',
        'desc': '我是描述一',

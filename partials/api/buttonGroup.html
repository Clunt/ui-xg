buttonGroup

Description

按钮组组件,可用于展示一组数据,实现类radio或checkbox效果。

Usage

<uix-button-group ng-model="string|object"
    [ bg-type="string" ]
    <button [btn-checkbox-true="string"]
        [btn-radio-val="string"]
        [btn-checkbox-false="string"]
        [name="string"]
        ></button>
</uix-button-group>

Restrict

  • 'AE'

Arguments

ParamTypeDefaultDetail
ngModel  设置按钮组绑定选中(radio:string|checkbox:object)数据对象,如果type为radio,则该取值对应设置的btnRadioVal,与ngModel值相同的btnRadioVal被选中,按钮间切换选择同时修改ngModel绑定数据;如果type为checkbox,则ngModel是一个对象,其键(key)是所有对应的子button的name值,而值(value)则是btnCheckboxFalse|btnCheckboxTrue(boolean类型)对应的值,btnCheckboxTrue表示key对应button被选中,btnCheckboxFalse则表示不选.如果不设置程序会报错.
bgType(optional) radio指定按钮组类型,取值radio(单选)或者checkbox(多选),默认为radio
btnCheckboxFalse(optional) falsecheckbox类型,设置子button不选时ngModel设置对象,对应的value值,默认为false
btnCheckboxTrue(optional) truecheckbox类型,设置子button选中时ngModel设置对象,对应的value值,默认为true
name(optional) 设置在子元素button上,主要用来标志每个子checkbox元素,设置ngModel的时候,通过name对应.
btnRadioVal(optional) 设置在子元素button上,主要用来根据ngModel的值判断是否选中,同时选中之后重新设置ngModel的值.如果设置type=radio,则必须设置
uncheckable(optional) false指定radio类型按钮组是否可取消选择,默认为false

Example

  1. 设置ngModel,默认为radio类型,分别设置子元素btnRadioVal

    interst:{{interst}}

  2. 设置ngModel,类型设置为checkbox类型,分别设置子元素name

    interstObj:{{interstObj}}

  3. 设置ngModel,类型设置为checkbox类型,分别设置子元素name和btn-checkbox-true,btn-checkbox-false

    setInterstObj:{{setInterstObj}}

<div ng-controller="buttonGroupDemoCtrl">

            </uix-button-group>
            <p> interst:{{interst}}</p>
        </li>
        <li>
            <p>设置ngModel,类型设置为checkbox类型,分别设置子元素name</p>
            <uix-button-group ng-model="interstObj" bg-type="checkbox">
                <label class="btn btn-default" name="firstHobby">跑步</label>
                <label class="btn btn-default" name="secondHobby">游泳</label>
                <label class="btn btn-default" name="thirdHobby">骑行</label>
            </uix-button-group>
            <p> interstObj:{{interstObj}}</p>
        </li>

                <label class="btn btn-default" name="firstHobby" btn-checkbox-true="'running'">跑步</label>
                <label class="btn btn-default" name="secondHobby" btn-checkbox-false="'noSwiming'">游泳</label>
                <label class="btn btn-default" name="thirdHobby">骑行</label>
            </uix-button-group>
            <p> setInterstObj:{{setInterstObj}}</p>
        </li>

angular.module('uixDemo').controller('buttonGroupDemoCtrl', ['$scope', function ($scope) {
    $scope.interst = 'running';
    $scope.interstObj = {
        firstHobby: true,
        secondHobby: false,
        thirdHobby: true
    };
    $scope.setInterstObj = {
        firstHobby: 'running',
        secondHobby: false,
        thirdHobby: true
    };
}]);
